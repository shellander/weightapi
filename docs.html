Weight Tracker API

Add Measurement

Endpoint: /add_measurement

Method: POST

Data Params:

{
    "user_id": [integer],
    "weight": [float],
    "date": [string (YYYY-MM-DD)]
}

Example using Python requests:

import requests

data = {
    "user_id": 1,
    "weight": 75.5,
    "date": "2023-04-25"
}
response = requests.post("http://localhost:5000/add_measurement", json=data)
print(response.json())

Edit Measurement

Endpoint: /edit_measurement/<measurement_id>

Method: PUT

URL Params: measurement_id=[integer]

Data Params:

{
    "weight": [float (optional)],
    "date": [string (YYYY-MM-DD) (optional)]
}

Example using Python requests:

import requests

measurement_id = 1
data = {
    "weight": 74.0,
    "date": "2023-04-26"
}
response = requests.put(f"http://localhost:5000/edit_measurement/{measurement_id}", json=data)
print(response.json())

Get Measurements for User

Endpoint: /get_measurements/<user_id>

Method: GET

URL Params: user_id=[integer]

Example using Python requests:

import requests

user_id = 1
response = requests.get(f"http://localhost:5000/get_measurements/{user_id}")
print(response.json())
